6. Things We Kept From VS Code (Intentionally)
Compatibility
Extension API Namespace: vscode
The entire extension API uses the vscode namespace. This is NOT branding — it's the API surface. Changing this would break every VS Code extension. We intentionally preserve it so users can install any extension from the marketplace.
Preserved in full source tree
Compatibility
Internal ENV Variables: VSCODE_*
Environment variables like VSCODE_IPC_HOOK_CLI, VSCODE_NODE_OPTIONS, VSCODE_DEV are internal Electron IPC variables. Not user-facing, but renaming would break upstream boot/WSL integration. Safe to leave as-is.
resources/*/bin/code.sh, code.cmd
